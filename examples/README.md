# Example Configurations

This directory contains example configurations for different use cases.

## Files

- `minimal-typescript.ts` - Minimal TypeScript configuration
- `minimal-toml.toml` - Minimal TOML configuration
- `minimal-lua.lua` - Minimal Lua configuration
- `minimal-vim.vim` - Minimal Vim script configuration
- `full-typescript.ts` - Full-featured TypeScript configuration
- `full-toml.toml` - Full-featured TOML configuration
- `neovim-ide.ts` - IDE-like setup for Neovim
- `vim-essentials.vim` - Essential plugins for Vim

## Usage

Copy the desired example to your configuration directory:

```bash
# For Neovim with TypeScript
cp examples/minimal-typescript.ts ~/.config/nvim/dpp.ts

# For Neovim with TOML
cp examples/minimal-toml.toml ~/.config/nvim/dpp.toml

# For Vim
cp examples/minimal-vim.vim ~/.config/vim/dpp.vim
```

Or use them as reference when creating your own configuration with `dpp init`.
